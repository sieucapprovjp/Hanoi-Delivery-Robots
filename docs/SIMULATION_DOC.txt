18. calculateEfficiencyScore: Computes a weighted performance metric for algorithm comparison.
---
[SIGNATURE]
- Input     : stats: dict[str, float | int] # Aggregate algorithm statistics.
- Output    : float # Normalized efficiency score.
- Calls     : None
- Constants : CONFIG.SIMULATION.EFFICIENCY_WEIGHTS
- Formulas  : denominator = totalPathCostKm + (weights.TIME * avgTimeMs) + (weights.NODES * avgNodes) + (weights.REROUTE * stats.rerouteCount) + 1, score = deliveriesCompleted / denominator

[LOGIC]
1. Calculate average execution time and node exploration per route.
2. Build a weighted cost denominator using path distance, computation time, nodes explored, and reroute frequency.
3. Return the efficiency ratio as completed deliveries per unit of weighted simulation cost.
