16. findShortestPath: Implements Dijkstra's algorithm to find optimal path considering traffic and roadblocks.
---
[SIGNATURE]
- Input     : startKey: str, endKey: str
- Output    : list[dict[str, float]] # Waypoint sequence
- Calls     : isRoadBlocked(), getSegmentTrafficPenalty()
- Constants : None
- Formulas  : new_dist = current_dist + (edge_dist * traffic_penalty)

[LOGIC]
1. Initialize Dijkstra distance table (Infinity) and predecessor map.
2. Maintain unvisited set and greedily process node with minimum known distance.
3. Evaluate neighbors of current node:
   - Skip if segment is actively blocked by roadblock.
   - Calculate candidate distance using base edge weight and real-time traffic penalty.
   - Update distance table and predecessor map if shorter path found.
4. Backtrack from target node to start to reconstruct waypoint sequence.

23. distanceToSegment: Calculates the shortest distance in meters from a point to a line segment.
---
[SIGNATURE]
- Input     : lat: float, lon: float, startLat: float, startLon: float, endLat: float, endLon: float
- Output    : float # Distance in meters
- Calls     : Math.cos(), Math.max(), Math.min(), Math.hypot()
- Constants : CONFIG.MAP.METERS_PER_DEGREE
- Formulas  : lon_scale = METERS_PER_DEGREE * cos(avg_lat); projection = dot_product / length_squared

[LOGIC]
1. Transform latitude/longitude coordinates into local Cartesian meter coordinates.
2. Account for longitude distortion based on average latitude of the points.
3. Use vector projection to find the point on the infinite line closest to the target.
4. Clamp the projection to segment bounds [0, 1] to find the closest point within the segment.
5. Return the Euclidean distance between target point and calculated segment point.

[LOGIC]

2. Apply the spherical Law of Cosines to calculate central angle between points.
3. Scale angle by Earth's radius to return distance in meters.
